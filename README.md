Welcome to Zeodic Signs, a website dedicated to exploring the 12 zodiac signs, their personalities, traits, and characteristics. This project is built using HTML, CSS, and JavaScript to provide an interactive and visually appealing experience for astrology enthusiasts.

Features

Overview of all 12 zodiac signs

Personality traits, strengths, and weaknesses of each sign

Interactive layout for easy navigation

Responsive design for both desktop and mobile

Zodiac Signs Included

Aries (March 21 – April 19)

Taurus (April 20 – May 20)

Gemini (May 21 – June 20)

Cancer (June 21 – July 22)

Leo (July 23 – August 22)

Virgo (August 23 – September 22)

Libra (September 23 – October 22)

Scorpio (October 23 – November 21)

Sagittarius (November 22 – December 21)

Capricorn (December 22 – January 19)

Aquarius (January 20 – February 18)

Pisces (February 19 – March 20)
